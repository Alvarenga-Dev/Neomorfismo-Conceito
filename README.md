# Neomorfismo

É uma forma elevada, feita exatamente do mesmo material que o fundo.

Esse efeito é bem fácil de se jogar com duas sombras, uma com valores negativos e outra com positivo. Mas, para que funcione, nosso plano de fundo não pode ser totalmente preto ou branco. Ele precisa de pelo menos um pouco de tonalidade para que as sombras escuras e claras sejam visíveis. Você pode usar qualquer tonalidade para o fundo, para que fique mais quente ou mais frio, dependendo da sua escolha. Mas sombras brancas e escuras precisam ser visíveis, mesmo que ligeiramente.
